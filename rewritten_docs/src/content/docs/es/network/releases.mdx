---
title: "Últimas Versiones de Libra2"
---

Cada versión de Libra2 se puede encontrar en el sitio de GitHub para [versiones de Libra2-core](https://github.com/libra2org/libra2-core/releases).
Cada versión está reflejada por las siguientes ramas de git:

- [Última Versión de Mainnet](https://github.com/libra2org/libra2-core/tree/mainnet)
- [Última Versión de Testnet](https://github.com/libra2org/libra2-core/tree/testnet)
- [Última Versión de Devnet](https://github.com/libra2org/libra2-core/tree/devnet)

Libra2 típicamente realiza múltiples versiones de devnet para cada versión de testnet y mainnet.
Por lo tanto, las versiones de devnet usan hashes de commit para el seguimiento en lugar de
números de versión. Las versiones de testnet y mainnet usualmente tienen una correlación
uno a uno, lo que significa que cada versión de testnet se convierte en mainnet.

Los hot fixes son excepciones que pueden ocurrir en mainnet para abordar problemas urgentes en
producción. Consulta el [Proceso de Versión de Libra2](https://github.com/libra2org/libra2-core/blob/main/RELEASE.md)
para más detalles.

## Actualizar nodos

Si eres un operador de nodo, [actualiza tus nodos con la nueva versión](/network/nodes/full-node/modify/update-fullnode-with-new-releases).

## Suscribirse a Anuncios de Versión

### Suscribirse vía GitHub

1. Ve al repositorio [libra2-labs/libra2-core](https://github.com/libra2org/libra2-core).
2. Abre el menú **Watch** y selecciona **Custom**.
3. Selecciona la casilla **Releases** y haz clic en **Apply**.

### Suscribirse vía [Libra2 Discord](https://discord.gg/libra2network)

Únete al servidor de Discord de Libra2 para interactuar con nosotros y nuestra comunidad. También publicamos próximas versiones en estos canales.

- [#mainnet-release](https://discord.com/channels/945856774056083548/1042502400507916349)
- [#testnet-release](https://discord.com/channels/945856774056083548/1025614160555413545)
- [#devnet-release](https://discord.com/channels/945856774056083548/956692649430093904)

### Suscribirse vía [Libra2 Twitter](https://x.com/Libra2Release)

Sigue [@Libra2Release](https://x.com/Libra2Release) en Twitter para obtener las últimas actualizaciones sobre nuestras próximas
versiones de mainnet y ser notificado cuando sea momento de actualizar tu nodo. Cada
pocos días, [@Libra2Release](https://x.com/Libra2Release) twitteará una cuenta regresiva para recordarte que actualices a
la última versión. _Nota: ¡No publicamos sobre hotfixes aquí!_

## Proceso de Versión de Libra2

Para entender cómo realizamos las versiones, revisa el [Proceso de Versión de Libra2](https://github.com/libra2org/libra2-core/blob/main/RELEASE.md).
