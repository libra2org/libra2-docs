如果你是参与 Libra2 网络的运营者,那么使用这份文档来执行运营者任务,比如部署验证节点和验证者全节点,将节点注册到 Libra2 社区平台,并执行验证.

<Aside type="note">
  **主网需要验证节点和验证者全节点** <br />
  为了参与到 Libra2 主网,你必须部署一个验证节点和一个验证者全节点.
</Aside>

## 部署节点并注册

<Steps>
  1. 阅读

     确保你的硬件,存储和网络资源满足节点要求.

  2. 部署节点

     按照提供的详细节点安装步骤 [**运行验证节点**](/zh/network/nodes/validator-node/deploy-nodes) 来部署一个验证节点和一个验证者全节点.
</Steps>

请注意,除非你通过下面的[质押池操作](/zh/network/nodes/validator-node/connect-nodes/staking-pool-operations)将它们添加到验证者集合中,否则你的节点将不会正确运行(不会同步,不会参与共识).

## 连接到 Libra2 网络

部署节点后,[连接到 Libra2 网络](/zh/network/nodes/validator-node/connect-nodes/connect-to-libra2-network).

## 设置质押池和委托池操作

将你的节点连接到 Libra2 网络后,建立[质押池操作](/zh/network/nodes/validator-node/connect-nodes/staking-pool-operations)以将你的节点添加到验证者集合中.

同样,为你的验证者委托的 APT 进行[委托池操作](/zh/network/nodes/validator-node/connect-nodes/delegation-pool-operations).你的节点将开始同步并参与共识.

## 确保你的节点在线

当您的节点部署并配置完成后,请确保它们达到[节点活跃性的要求](/zh/network/nodes/validator-node/verify-nodes/node-liveness-criteria).
