按照以下步骤快速开始.

<Steps>
  1. 初始化

     运行以下命令使用 `hello-blockchain` 模板初始化包:

     ```shellscript filename="Terminal"
     aptos move init --name hello_blockchain --template hello-blockchain
     ```

  2. 开始构建

     模板在 `sources` 下创建了一个 `hello_blockchain.move` 文件来帮助你开始.

     {/* <RemoteCodeblock permalink="https://github.com/aptos-labs/aptos-core/blob/afd3706c17bcccfb39a9d6059aecbfa648ed295d/aptos-move/move-examples/hello_blockchain/sources/hello_blockchain.move#L1-L64"></RemoteCodeblock> */}

  3. 查看所有模板

     运行以下命令查看所有模板(并获取初始化包的一般帮助):

     ```shellscript
     aptos move init --help
     ```
</Steps>
