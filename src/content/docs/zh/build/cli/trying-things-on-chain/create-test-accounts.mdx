title: "使用 Libra2 CLI 创建测试账户和发送交易"

<Aside type="note" emoji="ℹ️">
  如果您尚未安装 Libra2 CLI,可以按照[这些步骤](/zh/build/cli)进行安装.
</Aside>

通常,要在链上创建一个新账户,您需要先生成密钥,然后为该账户充值.在开发网上,您可以通过请求拥有测试 Libra2 代币的"水龙头"账户将它们发送到您的账户来为新账户提供资金.在测试网上,您可以在[铸币页面](/zh/network/faucet)进行铸币.

使用 CLI,您可以使用以下命令生成并为测试账户注资:

```shellscript filename="Terminal"
libra2 init --profile <您的配置文件名称>
```

一旦您拥有了一个有资金的账户,您可以使用 `transfer` 命令在账户之间发送代币,如下所示:

```shellscript filename="Terminal"
libra2 account transfer --account superuser --amount 100
```

<Aside type="note" emoji="ℹ️">
  如果你希望对生成的凭证(credentials)的具体格式有更精细的控制,可以用以下命令来代替 `libra2 init`:

  1. `libra2 key generate --vanity-prefix 0x<your-prefix>`
  2. `libra2 account fund-with-faucet --account <your-newly-generated-account-address>`

  但请注意,地址和密钥是不同的.
</Aside>
