title: "安装 Move Prover"

import { Aside } from '@astrojs/starlight/components';

如果您想使用 [Move Prover](/zh/build/smart-contracts/prover),请在[安装 CLI 二进制文件](/zh/build/cli/setup-cli/.)后安装 Move Prover 依赖项.
安装 Prover 依赖项有两种方式.

## 通过 Libra2 CLI 安装(推荐)

1. [安装最新的 Libra2 CLI 二进制文件](/zh/build/cli/install-cli/install-cli-mac).

2. 执行命令 `libra2 update prover-dependencies`.

<Aside type="note">
  安装后环境变量 `BOOGIE_EXE` 和 `Z3_EXE` 将自动设置.请确保它们在当前环境中生效.
</Aside>

## 通过 `libra2-core` 安装(不推荐)

1. 参见[从源代码构建 Libra2](/zh/network/nodes/building-from-source)

2. 然后,在检出的 libra2-core 目录中,安装额外的 Move 工具:

   <details>
     <summary>Linux / macOS</summary>

     1. 打开终端会话.
     2. 运行开发设置脚本来准备您的环境:`./scripts/dev_setup.sh -yp`
     3. 更新您当前的 shell 环境:`source ~/.profile`

     <Aside type="note" emoji="ℹ️">
       `dev_setup.sh -p` 使用环境变量更新您的 `~./profile` 以支持已安装的 Move Prover 工具.您可能需要为您的 shell 设置 `.bash_profile` 或 `.zprofile` 或其他设置文件.
     </Aside>
   </details>

   <details>
     <summary>Windows</summary>

     1. 以管理员身份打开 PowerShell 终端.
     2. 运行开发设置脚本来准备您的环境:`PowerShell -ExecutionPolicy Bypass -File ./scripts/windows_dev_setup.ps1 -y`
   </details>

安装后,您可以运行 Move Prover 来验证一个[示例](https://github.com/libra2org/libra2-core/tree/main/libra2-move/move-examples/hello_prover):

```shellscript filename="Terminal"
libra2 move prove --package-dir libra2-move/move-examples/hello_prover/
```

## 故障排除

如果在运行命令时遇到如下错误,请检查您的 Libra2 CLI 版本或验证您使用的是正确的 `libra2` 工具,特别是如果您安装了多个版本.

