title: "在 Linux 上安装 Aptos CLI"
---

对于 Linux 系统,安装 Aptos CLI 工具最简单的方式是通过 shell 脚本.如果脚本方式不可行,您也可以通过下载预编译二进制文件进行手动安装.不过一般不推荐预编译二进制文件的方式,因为更新过程非常手动化.

# 通过脚本安装

<Steps>
  1. 在终端中使用以下任一命令:

     ```shellscript filename="Terminal"
     curl -fsSL "https://aptos.dev/scripts/install_cli.sh" | sh
     ```

     或使用等效的 `wget` 命令:

     ```shellscript filename="Terminal"
     wget -qO- "https://aptos.dev/scripts/install_cli.sh" | sh
     ```

     <Aside type="caution">
       如果运行 CLI 时遇到 `Illegal instruction` 错误,可能是由于您的 CPU 不支持 SIMD 指令.
       特别是对于较旧的非 SIMD 处理器或在 ARM Mac 上的 Ubuntu x86\_64 docker 容器,您可能需要改用以下命令来跳过 SIMD 指令:

       ```shellscript filename="Terminal"
         curl -fsSL "https://aptos.dev/scripts/install_cli.sh" | sh -s -- --generic-linux
       ```
     </Aside>

  2. (可选)将 Aptos CLI 添加到 PATH 目录或直接添加到 PATH 会很有帮助.

     - 将目录添加到 PATH 的步骤取决于您使用的 shell.
     - 您可以运行 `echo $SHELL` 来打印机器的默认 shell,然后搜索针对该 shell 添加目录到 PATH 的具体步骤.

  3. 打开新终端并运行 aptos help 来验证安装是否成功

     - 您应该能看到 CLI 可运行的命令列表.
     - 未来这将是一个有用的资源,可以详细了解每个命令的工作原理.
</Steps>

<Aside type="note" emoji="ℹ️">
  如果想将 Aptos CLI 更新到最新版本,可以运行 `aptos update`.
</Aside>

# 通过包管理器安装(可选)

<Aside type="note">
  通过包管理器安装 Aptos 后,未来请通过相同的包管理器进行更新.
</Aside>

### Arch Linux

#### 通过 AUR (Arch 用户仓库) 安装

```shellscript filename="Terminal"
git clone https://aur.archlinux.org/aptos-bin.git
cd aptos-bin
makepkg -si
```

或使用 AUR 助手如 `yay`:

```shellscript filename="Terminal"
yay -S aptos-bin
```

# 通过预编译二进制文件安装(备用方法)

<Steps>
  1. 访问 .

  2. 点击最新版本的 "Assets" 可展开菜单查看预编译二进制文件.

  3. 下载适用于 Linux 的 zip 文件.

     1. 文件名类似:`aptos-cli-<version>-Linux-x86_64.zip` 或 `aptos-cli-<version>-Linux-aarch64.zip`.
     2. 确保选择与您计算机架构匹配的 zip 文件(x86\_64 对应 Intel/AMD,aarch64 对应 ARM).
     3. 下载时可能需要忽略关于该文件可疑的警告.

  4. 解压下载的文件.

  5. 移动将提取的 Aptos 二进制文件放入您偏好的文件夹.

  6. 打开终端并导航至您偏好的文件夹.

  7. 通过运行 chmod +x ~/aptos 使 ~/aptos 成为可执行文件.

  8. 运行 ~/aptos help 验证已安装版本是否正常工作.

     您应该会看到所有 CLI 命令的使用说明.这些说明在未来尝试理解如何使用特定命令时会很有帮助.

  9. (可选)将 Aptos CLI 添加到 PATH 环境变量的某个文件夹,或直接添加到 PATH 会很有帮助.

     - 将文件夹添加到 PATH 的步骤取决于您使用的 shell.
     - 您可以运行 `echo $SHELL` 打印您机器的默认 shell,然后通过搜索了解为该 shell 添加文件夹到 PATH 的具体步骤.
</Steps>

<Aside type="note" emoji="ℹ️">
  当使用预编译二进制文件安装方式时,你可以通过删除现有安装,然后重新执行安装步骤来更新 Aptos CLI.
</Aside>
