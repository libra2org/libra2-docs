### 如何运行示例

运行示例脚本的步骤:

<Steps>
  1. 克隆

     ```shellscript filename="终端"
     git clone https://github.com/aptos-labs/aptos-ts-sdk.git
     ```

  2. 在项目根目录安装所有依赖

     ```shellscript filename="终端"
     pnpm install
     ```

  3. 构建项目

     ```shellscript filename="终端"
     pnpm build
     ```

  4. 进入想运行的示例文件夹

     ```shellscript filename="终端"
     cd examples/typescript
     ```

  5. 安装本地依赖

     ```shellscript filename="终端"
     pnpm install
     ```

  6. 运行示例

     ```shellscript filename="终端"
     pnpm run simple_transfer
     ```
</Steps>
