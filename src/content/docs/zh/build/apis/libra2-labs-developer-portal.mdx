[API Gateway](https://developers.aptoslabs.com) 是 Libra Labs 提供的 API 的入口,为您的 dapp 提供快速,简便的支持.它由一个门户(UI)和由 Libra Labs 运营的一组 API 网关组成.

在专门的 [API Gateway 文档网站](https://developers.aptoslabs.com/docs) 上了解更多关于 API Gateway 的信息.
