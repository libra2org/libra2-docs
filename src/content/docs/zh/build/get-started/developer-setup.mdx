根据开发类型的不同,以下是配置环境的简易指南.

<Tabs>
  {/* 前端开发 */}

  <TabItem label="前端开发">
    <Steps>
      1. 初始化前端项目

         以下是几种流行框架的示例:

         <Tabs>
           <TabItem label="Next.js">
             ```shellscript filename="终端"
             pnpx create-next-app@latest
             ```
           </TabItem>

           <TabItem label="Vite (TS)">
             ```shellscript filename="终端"
             pnpx create vite my-aptos-app --template react-ts
             ```
           </TabItem>
         </Tabs>

      2. 安装 @aptos-labs/ts-sdk

         ```shellscript npm2yarn
         npm i @aptos-labs/ts-sdk
         ```

      3. 配置 TS SDK

         <CardGrid>
           <LinkCard href="/zh/build/sdks/ts-sdk/quickstart" title="TS SDK 快速入门" description="了解如何设置账户、网络、使用水龙头、发送/模拟交易等" />
         </CardGrid>

      4. 开始构建应用!

         现在已完成前端集成 Aptos 的开发环境配置.
         查看我们其他能优化开发流程的工具

         <CardGrid>
           <LinkCard href="/zh/build/indexer" title="索引器" description="高效查询链上状态，如余额、交易活动、代币数据等" />

           <LinkCard href="https://github.com/aptos-labs/aptos-ts-sdk/tree/main/examples/typescript" title="TS SDK 示例" description="20+ 个 TS SDK 使用示例" target="_blank" />

           <LinkCard href="https://build.aptoslabs.com/" title="Aptos Build" description="遇到 Fullnode API/索引器速率限制？可在此获取API密钥" target="_blank" />
         </CardGrid>
    </Steps>
  </TabItem>

  {/* 智能合约 */}

  <TabItem label="智能合约">
    <Steps>
      1. 安装 CLI

         <CardGrid>
           <LinkCard href="/zh/build/cli" title="Aptos CLI" description="Aptos CLI 安装指南" />
         </CardGrid>

      2. 配置编辑器或 IDE

         为您的编辑器添加以下扩展以简化 Move 开发

         <Tabs>
           <TabItem label="JetBrains IDE">
             <CardGrid>
               <LinkCard
                 href="https://plugins.jetbrains.com/plugin/14721-move-language"
                 title="Move 语言插件"
                 description="语言服务器和语法高亮插件为
         JetBrains 系列 IDE（如 CLion、Rust Rover、WebStorm、
         IntelliJ）"
                 target="_blank"
               />
             </CardGrid>
           </TabItem>

           <TabItem label="VSCode">
             <CardGrid>
               <LinkCard href="https://marketplace.visualstudio.com/items?itemName=MoveBit.aptos-move-analyzer" title="Aptos Move 分析器" description="适用于 VSCode 的语言服务器和语法高亮工具" target="_blank" />
             </CardGrid>
           </TabItem>
         </Tabs>

      3. 创建智能合约

         导航到您的应用目录并通过以下命令初始化一个新的智能合约:

         ```shellscript filename="Terminal"
         aptos move init --name my_todo_list
         ```

      4. 构建,编译和部署智能合约!

         用于 Aptos 智能合约开发的设置现已完成.
         更多信息请查看下方 Dapp 教程链接

         <CardGrid>
           <LinkCard href="/zh/build/guides/build-e2e-dapp/1-create-smart-contract#what-is-a-movetoml-file" title="智能合约创建指南" description="一个简易的待办事项列表指南，介绍如何使用 Move 设置智能合约" />
         </CardGrid>
    </Steps>
  </TabItem>

  {/* create-aptos-dapp */}

  <TabItem label="创建 Aptos Dapp">
    <Steps>
      1. 安装 create-aptos-dapp

         运行以下命令,几秒内即可从模板安装一个 dApp:

         <Tabs>
           { /* npx */ }

           <TabItem label="npx">
             ```shellscript filename="Terminal"
             npx create-aptos-dapp@latest
             ```
           </TabItem>

           { /* pnpx */ }

           <TabItem label="pnpx">
             ```shellscript filename="Terminal"
             pnpx create-aptos-dapp@latest
             ```
           </TabItem>
         </Tabs>

      2. 跟随提示操作

         根据 CLI 提示选择您新 dApp 的名称,[模板](/zh/build/create-aptos-dapp#templates) 和网络.

         ![cad](~/images/cad-video.gif)

      3. 开始构建并定制您的新 dApp!

         导航到您的新项目并在您喜欢的 IDE 中打开以继续开发!

         按照生成的 `README.md` 文件进行后续步骤.

      4. 继续阅读

         <CardGrid>
           <LinkCard href="/zh/build/create-aptos-dapp" title="创建 Aptos Dapp" description="获取关于该工具的更多信息" />

           <LinkCard href="/zh/build/create-aptos-dapp#templates" title="模板" description="浏览预制模板" />

           <LinkCard href="/zh/build/create-aptos-dapp/faq" title="常见问题" description="获取常见问题和解决方案" />
         </CardGrid>
    </Steps>
  </TabItem>
