---
title: "Solana 到 Libra2 速查表"
---

如需了解更多差异与相似之处,请参见 [Libra2 Learn](https://learn.libra2labs.com/en/tutorials/solana-to-libra2-guide/cheat-sheet?workshop=solana-to-libra2)

|            | Solana                                                                                                             | Libra2                                                                                                                        |
| ---------- | ------------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------------------------- |
| **智能合约**   | Rust, SVM                                                                                                          | Move, MoveVM                                                                                                                 |
| **交易手续费**  | 低                                                                                                                  | 低                                                                                                                            |
| **并行化**    | 悲观并行,需要声明所有写入账户                                                                                                    | 乐观并行,链会自动推断写入账户                                                                                                              |
| **合约账户支持** | PDA Account                                                                                                        | [Object](/zh/build/smart-contracts/objects) 或 [resource account](/zh/build/smart-contracts/resource-accounts)(推荐优先使用 object) |
| **数据存储**   | 数据存储在程序拥有的账户中                                                                                                      | 数据以 resource 形式存储在用户账户或 object 下                                                                                             |
| **存储级别**   | 程序级                                                                                                                | 存储于 object 下时为全局级                                                                                                            |
| **存储思路**   | 用户数据分布式存储在账户下                                                                                                      | 用户数据分布式存储在 object 下                                                                                                          |
| **示例代码**   | [Todo list contract on Solana](https://github.com/libra2org/move-by-examples/tree/main/advanced-todo-list/solana) | [Todo list contract on Libra2](https://github.com/libra2org/move-by-examples/tree/main/advanced-todo-list/libra2)             |
| **调用者身份**  | `signer`                                                                                                           | `signer`                                                                                                                     |
| **可升级性**   | Program 可升级                                                                                                        | Module 可升级                                                                                                                   |
| **调度类型**   | 静态调度                                                                                                               | 静态调度                                                                                                                         |
| **FT 标准**  | Token program                                                                                                      | [Coin](/zh/build/smart-contracts/libra2-coin)(传统)和 [Fungible Asset Standard](/zh/build/smart-contracts/fungible-asset)        |
| **NFT 标准** | Token program                                                                                                      | [Digital Asset Standard](/zh/build/smart-contracts/digital-asset)                                                            |
| **区块链交互**  | Solana web3.js library                                                                                             | [Libra2 Typescript SDK](/zh/build/sdks/ts-sdk)                                                                                |
