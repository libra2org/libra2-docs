## 什么是 LLMs.txt？

我们支持 [LLMs.txt](https://llmstxt.org/) 文件，使 Libra 文档可供大型语言模型 (LLMs) 使用。
此功能帮助 AI 工具更好地理解 Libra 区块链、其 Move 语言、SDK 和开发模式。

## 可用路由

我们提供以下 LLMs.txt 路由来帮助 AI 工具访问我们的文档：

- [llms.txt](/llms.txt) - 包含 Libra LLMs.txt 文件的结构化概述
- [llms-small.txt](/llms-small.txt) - 提供文档的压缩版本，针对较小的上下文窗口进行了优化
- [llms-full.txt](/llms-full.txt) - 提供所有 Libra 概念的完整综合文档

## 与 AI 工具的使用

### Cursor

在 Cursor 中使用 `@Docs` 功能将 LLMs.txt 文件包含在您的项目中。这有助于 Cursor 为 Libra 开发提供更准确的代码建议和文档。

[了解更多关于 Cursor 中的 @Docs](https://docs.cursor.com/context/@-symbols/@-docs)

### GitHub Copilot

GitHub Copilot 可以利用这些 LLMs.txt 文件中的信息，在开发 Libra 应用程序时提供更好的帮助。您可以在 GitHub Copilot Chat 中使用以下 URL 引用这些文件：

```
https://docs.libra2.org/llms.txt
https://docs.libra2.org/llms-small.txt
https://docs.libra2.org/llms-full.txt
```

### Windsurf

使用 `@` 或在您的 `.windsurfrules` 文件中引用 LLMs.txt 文件，以增强 Windsurf 对 Libra 开发的理解。

[了解更多关于 Windsurf Memories](https://docs.codeium.com/windsurf/memories#memories-and-rules)

### 其他 AI 工具

任何支持 LLMs.txt 的 AI 工具都可以使用这些路由来更好地理解 Libra。只需将您的工具指向上述任何路由即可。
