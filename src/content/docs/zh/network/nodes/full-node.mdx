title: "运行公共全节点"
---

import { Aside } from '@astrojs/starlight/components';

你可以运行自己的公共全节点(PFN),以同步 Libra 区块链的状态并同步最新内容.PFN 通过与其他 Libra 验证全节点(VFN)和公共全节点(PFN)同步,复制区块链的整个状态.任何人都可以运行 PFN.本节解释了如何部署 PFN 并连接到 Libra 网络.你可以在[区块链详解](/zh/network/blockchain)部分了解更多关于不同类型的节点.

<Aside type="note">
  **默认连接到主网**<br />
  如果你按照本文档中的默认设置操作,你的 PFN 将连接到 Libra 主网.要连接到不同的 Libra 网络,如 devnet 或 testnet,请确保你有正确的 docker 镜像标签,或者如果你想直接构建二进制文件,则需要正确的源代码分支.

  你可以在这里找到所有网络的创世区块和路标文件 ➜ https://github.com/libra2org/aptos-networks.
</Aside>
