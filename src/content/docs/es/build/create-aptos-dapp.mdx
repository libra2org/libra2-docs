`create-aptos-dapp` construye un proyecto plantilla para desarrolladores de dapps para crear fácilmente un front-end y un contrato inteligente en la red de Aptos.

## ¿Por qué usar create-aptos-dapp?

- **Configuración con Plantillas**: `create-aptos-dapp` genera plantillas de dapps de extremo a extremo predefinidas y archivos de configuración para ti. Ahorra la configuración manual de la estructura del proyecto, que puede ser consumidora de tiempo y propensa a errores.
- **Directorio de Contratos:** `create-aptos-dapp` genera un directorio `contract` que incluye la estructura básica para módulos de contratos inteligentes Move.
- **Mejores Prácticas**: `create-aptos-dapp` incorpora mejores prácticas y recomendaciones de estructura para desarrollar en la red de Aptos.
- **Comandos Move Integrados**: `create-aptos-dapp` incluye comandos integrados para tareas comunes, como inicializar el compilador Move, compilar y publicar contratos inteligentes en la cadena.

## Prerrequisitos

- [node y npm](https://nodejs.org/en) (npm ≥ 5.2.0)
- [Python 3.6+](https://www.python.org/)

## Plantillas

`create-aptos-dapp` te proporciona plantillas de dapps de extremo a extremo premade, es decir, una dapp lista con configuraciones y una hermosa UI para comenzar a crear una dapp en Aptos.

Los objetivos de las plantillas son:

1. Familiarizar a los usuarios con diferentes Estándares de Aptos teniendo ejemplos de plantillas de dapps de extremo a extremo.
2. Educar a los usuarios sobre cómo construir una dapp en Aptos desde la capa de front-end hasta la capa de contratos inteligentes y cómo todo lo que está en medio.
3. Proporcionar a los usuarios plantillas premade para desplegar rápidamente dapps simples

## Herramientas que `create-aptos-dapp` utiliza

- Framework React
- Herramienta de desarrollo Vite
- shadcn/ui + tailwind para estilos
- SDK de TS de Aptos
- Adaptador de Wallet de Aptos
- Comandos Move basados en Node
