La red de Blockchain de Libra2 puede ser accedida por varias APIs, dependiendo de tu caso de uso.

## Nodo Completo de Libra2

Esta API - integrada en Nodos Completos - proporciona una forma simple, de baja latencia, pero de bajo nivel de _leer_ estado y
_enviar_ transacciones a la Blockchain de Libra2. También soporta simulación de transacciones.

El código de cada una de las APIs mencionadas anteriormente es de código abierto en [GitHub](https://github.com/libra2org/libra2-core). Como
tal, cualquiera puede operar estas APIs y muchos operadores y constructores independientes en todo el mundo eligen hacerlo.

### Despliegues de API Operados por Libra2 Labs

[Libra2 Labs](https://libra2labs.com) opera un despliegue de estas APIs en nombre de [Libra2 Foundation](https://libra2foundation.org/)
para cada [Red de Libra2](/network/nodes/networks) y las hace disponibles para consumo público.

Estas APIs permiten acceso limitado por IP sin una clave de API (acceso anónimo). Para obtener límites de tasa mucho más altos puedes registrarte para una cuenta de [Libra2 Build](https://build.libra2labs.com/).
