## Instalando el SDK de Go

Libra proporciona un SDK oficial de Go en
el repositorio [libra2-go-sdk GitHub](https://github.com/libra2org/libra2-go-sdk).
Para usar el SDK de Go, obtén el paquete principal aquí:

```shellscript
go get github.com/libra2org/libra2-go-sdk
```
