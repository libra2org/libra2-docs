## Instalación

Aptos proporciona un SDK oficial de Rust con soporte ligero en el
repositorio [Aptos-core GitHub](https://github.com/aptos-labs/aptos-core/tree/main/sdk).
Para usar el SDK de Rust, agrega la siguiente dependencia y parches en el
repositorio git directamente en tu `Cargo.toml`, así:

```toml filename="Cargo.toml"
[dependencies]
aptos-sdk = { git = "https://github.com/aptos-labs/aptos-core", branch = "devnet" }

[patch.crates-io]
merlin = { git = "https://github.com/aptos-labs/merlin" }
x25519-dalek = { git = "https://github.com/aptos-labs/x25519-dalek", branch = "zeroize_v1" }
```

También debes crear un archivo `.cargo/config.toml` con este contenido:

```toml filename=".cargo/config.toml"
[build]
rustflags = ["--cfg", "tokio_unstable"]
```

El código fuente para el SDK oficial de Rust está disponible en el
[repositorio GitHub aptos-core](https://github.com/aptos-labs/aptos-core/tree/main/sdk).
