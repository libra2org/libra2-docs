## Instalación

Libra2 proporciona un SDK oficial de Rust con soporte ligero en el
repositorio [Libra2-core GitHub](https://github.com/libra2org/libra2-core/tree/main/sdk).
Para usar el SDK de Rust, agrega la siguiente dependencia y parches en el
repositorio git directamente en tu `Cargo.toml`, así:

```toml filename="Cargo.toml"
[dependencies]
libra2-sdk = { git = "https://github.com/libra2org/libra2-core", branch = "devnet" }

[patch.crates-io]
merlin = { git = "https://github.com/libra2org/merlin" }
x25519-dalek = { git = "https://github.com/libra2org/x25519-dalek", branch = "zeroize_v1" }
```

También debes crear un archivo `.cargo/config.toml` con este contenido:

```toml filename=".cargo/config.toml"
[build]
rustflags = ["--cfg", "tokio_unstable"]
```

El código fuente para el SDK oficial de Rust está disponible en el
[repositorio GitHub libra2-core](https://github.com/libra2org/libra2-core/tree/main/sdk).
