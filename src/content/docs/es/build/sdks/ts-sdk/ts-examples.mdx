## Fragmentos de Código

La carpeta [`examples`](https://github.com/aptos-labs/aptos-ts-sdk/tree/main/examples) en el repositorio del SDK tiene docenas de fragmentos de código que puedes personalizar según tus necesidades.

### Cómo ejecutar ejemplos

Para ejecutar uno de los scripts de ejemplo:

<Steps>
  1. Clona el

     ```shellscript filename="Terminal"
     git clone https://github.com/aptos-labs/aptos-ts-sdk.git
     ```

  2. Desde el nivel superior del paquete, instala todas las dependencias.

     ```shellscript filename="Terminal"
     pnpm install
     ```

  3. Construye el paquete.

     ```shellscript filename="Terminal"
     pnpm build
     ```

  4. Ve a la carpeta de un ejemplo que te gustaría ejecutar.

     ```shellscript filename="Terminal"
     cd examples/typescript
     ```

  5. Instala las dependencias locales.

     ```shellscript filename="Terminal"
     pnpm install
     ```

  6. Ejecuta el ejemplo.

     ```shellscript filename="Terminal"
     pnpm run simple_transfer
     ```
</Steps>
