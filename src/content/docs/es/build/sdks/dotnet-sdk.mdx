Integra capacidades Web3 de Aptos dentro de tus aplicaciones .NET. El objetivo de este SDK es proporcionar un conjunto de herramientas
para que los desarrolladores construyan aplicaciones multiplataforma a través de motores de juegos y plataformas compatibles.

**Características Soportadas**

- Codificación y decodificación de Serialización Canónica Binaria (BCS)
- Soporte para firmantes Ed25519, SingleKey, MultiKey y Keyless
- Utilidades para construcción, firma y envío de transacciones
- Abstracciones sobre las APIs del Nodo Completo e Indexador de Aptos
- Soporte para Nombres de Aptos (ANS) para resolución y búsqueda

## Instalación

El SDK de .NET está disponible en [NuGet](https://www.nuget.org/packages/Aptos).

Puedes instalar el SDK de .NET usando el siguiente comando:

```shellscript
dotnet add package Aptos
```
