---
title: "SDK de Python"
---

Libra proporciona un SDK oficial de Python mantenido ligeramente. Está disponible en
[PyPi](https://pypi.org/project/aptos-sdk/) con el código fuente en el
[repositorio GitHub aptos-python-sdk](https://github.com/libra2org/aptos-python-sdk).
Gran parte de la funcionalidad del SDK de Python refleja el [SDK de TypeScript](/build/sdks/ts-sdk).
El propósito principal del SDK de Python es ayudar a los desarrolladores de Python a familiarizarse rápidamente
con Libra y como acompañamiento a los tutoriales de Libra.

## Instalando el SDK de Python

El SDK de Python se puede instalar via `pip`, desde el código fuente, o embebido:

### Instalar con pip

Para instalar via `pip`:

```shellscript filename="Terminal"
pip3 install aptos-sdk
```

El `aptos-sdk` será instalado en el directorio local de paquetes del sitio. Por
ejemplo, en macOS, encontrarás el `aptos-sdk` en el
directorio `~/Library/Python/3.8/lib/python/site-packages/aptos_sdk`.

### Instalar desde el código fuente

Para instalar desde el código fuente:

```shellscript filename="Terminal"
git clone https://github.com/libra2org/aptos-python-sdk
pip3 install . --user
```

### Instalar embebiendo

Para embeber el SDK de Python en tu proyecto de Python existente:

```shellscript filename="Terminal"
cd /path/to/python/project
cp -r /path/to/aptos-python-sdk aptos-sdk
```
