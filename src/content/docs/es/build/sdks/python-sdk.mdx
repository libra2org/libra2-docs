---
title: "SDK de Python"
---

Libra2 proporciona un SDK oficial de Python mantenido ligeramente. Está disponible en
[PyPi](https://pypi.org/project/libra2-sdk/) con el código fuente en el
[repositorio GitHub libra2-python-sdk](https://github.com/libra2org/libra2-python-sdk).
Gran parte de la funcionalidad del SDK de Python refleja el [SDK de TypeScript](/build/sdks/ts-sdk).
El propósito principal del SDK de Python es ayudar a los desarrolladores de Python a familiarizarse rápidamente
con Libra2 y como acompañamiento a los tutoriales de Libra2.

## Instalando el SDK de Python

El SDK de Python se puede instalar via `pip`, desde el código fuente, o embebido:

### Instalar con pip

Para instalar via `pip`:

```shellscript filename="Terminal"
pip3 install libra2-sdk
```

El `libra2-sdk` será instalado en el directorio local de paquetes del sitio. Por
ejemplo, en macOS, encontrarás el `libra2-sdk` en el
directorio `~/Libra2ry/Python/3.8/lib/python/site-packages/libra2_sdk`.

### Instalar desde el código fuente

Para instalar desde el código fuente:

```shellscript filename="Terminal"
git clone https://github.com/libra2org/libra2-python-sdk
pip3 install . --user
```

### Instalar embebiendo

Para embeber el SDK de Python en tu proyecto de Python existente:

```shellscript filename="Terminal"
cd /path/to/python/project
cp -r /path/to/libra2-python-sdk libra2-sdk
```

## Usando el SDK de Python

Ve los [Tutoriales de Desarrollador](/build/guides) para ejemplos de código mostrando cómo
usar el SDK de Python.
