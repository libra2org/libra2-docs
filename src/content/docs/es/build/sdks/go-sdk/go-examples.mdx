## Fragmentos de Código

La carpeta [`examples`](https://github.com/aptos-labs/aptos-go-sdk/tree/main/examples)
en el repositorio del SDK tiene muchos fragmentos de código que puedes personalizar según tus necesidades.

### Cómo ejecutar ejemplos

Para ejecutar uno de los scripts de ejemplo:

<Steps>
  1. Clona el

     ```shellscript filename="Terminal"
     git clone https://github.com/aptos-labs/aptos-go-sdk.git
     ```

  2. Desde el nivel superior del paquete, instala todas las dependencias.

     ```shellscript filename="Terminal"
     go install
     ```

  3. Construye el paquete.

     ```shellscript filename="Terminal"
     go build ./...
     ```

  4. Ejecuta un ejemplo

     ```shellscript filename="Terminal"
     go run examples/transfer_coin/main.go
     ```
</Steps>
