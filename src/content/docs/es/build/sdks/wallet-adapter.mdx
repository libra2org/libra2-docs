Hay dos estándares de adaptador de wallet en el ecosistema de Aptos:

1. [Adaptador de Wallet de Aptos](#aptos-wallet-adapter) por Aptos Labs
2. [OKX Connect](#okx-connect) por OKX

## Adaptador de Wallet de Aptos

El Adaptador de Wallet de Aptos por Aptos Labs proporciona una interfaz única para que las dapps
de Aptos y las wallets de Aptos se comuniquen entre sí.

Para desarrolladores de dapps, esto significa que puedes conectarte a cualquier wallet de Aptos que esté
integrada con el Adaptador de Wallet sin necesidad de escribir código personalizado para cada
wallet. Esto se describe en [AIP-62](https://github.com/aptos-foundation/AIPs/blob/main/aips/aip-62.md),
que define el Estándar de Wallet Moderno y la autodetección de wallets.

Para proveedores de wallets de Aptos, integrar con AIP-62 significa que tu wallet será
compatible con cualquier dapp que use el Adaptador de Wallet.

### Para Dapps de Aptos

Sigue la [Guía del Adaptador de Wallet para Constructores de Dapps](/build/sdks/wallet-adapter/dapp) sobre cómo usar el
Adaptador de Wallet (via el [paquete Wallet Adapter React](https://github.com/aptos-labs/aptos-wallet-adapter/tree/main/packages/wallet-adapter-react)).

### Para Proveedores de Wallets de Aptos

Sigue una de estas guías para cómo implementar un plugin del Adaptador de Wallet que
las dapps puedan conectar:

1. Para [Wallets de Extensión de Navegador](/build/sdks/wallet-adapter/browser-extension-wallets) (ej. [Petra](https://chromewebstore.google.com/detail/petra-aptos-wallet/ejjladinnckdgjemekebdpeokbikhfci?hl=en))
2. Para [Wallets SDK](/build/sdks/wallet-adapter/wallets) (ej. [AptosConnect](https://aptosconnect.app))

## OKX Connect

El adaptador OKX Connect proporciona una interfaz para que las dapps de Aptos se conecten a la wallet
de OKX y otras wallets que soporten el estándar OKX Connect. Puedes encontrar
más información sobre OKX Connect para dapps de Aptos en la [documentación de OKX Connect](https://web3.okx.com/build/dev-docs/sdks/app-connect-aptos)
