import { CardGrid, LinkCard, Steps, TabItem, Tabs } from '@astrojs/starlight/components';

Aquí hay una forma fácil de configurar tu entorno dependiendo del tipo de desarrollo.

<Tabs>
  {/* Frontend */}

  <TabItem label="Frontend">
    <Steps>
      1. Inicializar Proyecto Frontend

         Aquí hay algunos ejemplos de opciones populares:

         <Tabs>
           <TabItem label="Next.js">
             ```shellscript filename="Terminal"
             pnpx create-next-app@latest
             ```
           </TabItem>

           <TabItem label="Vite (TS)">
             ```shellscript filename="Terminal"
             pnpx create vite my-libra2-app --template react-ts
             ```
           </TabItem>
         </Tabs>

      2. Instalar @libra2-core/ts-sdk

         ```shellscript npm2yarn
         npm i @libra2-core/ts-sdk
         ```

      3. Configurar TS SDK

         <CardGrid>
           <LinkCard
             href="/es/build/sdks/ts-sdk/quickstart"
             title="Inicio Rápido TS SDK"
             description="Ve cómo configurar tu cuenta, red, usar el
         faucet, enviar / simular transacciones, y más"
           />
         </CardGrid>

      4. ¡Construye tu app!

         La configuración de desarrollador para usar Libra en tu frontend ahora está completa.
         Revisa nuestras otras herramientas que agilizan el proceso de desarrollo

         <CardGrid>
           <LinkCard
             href="/es/build/indexer"
             title="Indexador"
             description="Consulta eficientemente el estado de la cadena como balances,
         actividad de transacciones, datos de tokens, y más"
           />

           <LinkCard
             href="https://github.com/libra2org/libra2-ts-sdk/tree/main/examples/typescript"
             title="Ejemplos TS SDK"
             description="20+ Ejemplos de cómo usar el TS
         SDK"
             target="_blank"
           />

           <LinkCard
             href="https://build.libra2labs.com/"
             title="Libra Build"
             description="¿Alcanzando límites de tasa para API de Nodo Completo / Indexadores? Obtén
         una Clave de API aquí"
             target="_blank"
           />
         </CardGrid>
    </Steps>
  </TabItem>

  {/* Smart Contract */}

  <TabItem label="Contrato Inteligente">
    <Steps>
      1. Instalar CLI

         <CardGrid>
           <LinkCard href="/es/build/cli" title="CLI de Libra" description="Instrucciones para cómo instalar la CLI de Libra" />
         </CardGrid>

      2. Configurar Editor o IDE

         Agrega las siguientes extensiones a tu editor de elección para hacer el desarrollo
         de Move más fácil

         <Tabs>
           <TabItem label="JetBrains IDE">
             <CardGrid>
               <LinkCard
                 href="https://plugins.jetbrains.com/plugin/14721-move-language"
                 title="Move Language"
                 description="Servidor de lenguaje y resaltador de sintaxis para
         IDEs de JetBrains como CLion, Rust Rover, WebStorm,
         IntelliJ"
                 target="_blank"
               />
             </CardGrid>
           </TabItem>

           <TabItem label="VSCode">
             <CardGrid>
               <LinkCard
                 href="https://marketplace.visualstudio.com/items?itemName=MoveBit.libra2-move-analyzer"
                 title="Libra Move Analyzer"
                 description="Servidor de lenguaje y resaltador de sintaxis para
         VSCode"
                 target="_blank"
               />
             </CardGrid>
           </TabItem>
         </Tabs>

      3. Crear Contrato Inteligente

         Navega a tu carpeta de aplicación e inicializa un nuevo contrato inteligente
         haciendo:

         ```shellscript filename="Terminal"
         libra2 move init --name my_todo_list
         ```

      4. ¡Construye, Compila y Despliega el Contrato Inteligente!

         La configuración de desarrollador para usar Libra para contratos inteligentes ahora está completa.
         Para más información ve el enlace al tutorial de Dapp abajo

         <CardGrid>
           <LinkCard
             href="/es/build/guides/build-e2e-dapp/1-create-smart-contract#what-is-a-movetoml-file"
             title="Guía de Crear Contrato Inteligente"
             description="Una guía fácil de lista de tareas para cómo configurar un contrato
         inteligente con Move"
           />
         </CardGrid>
    </Steps>
  </TabItem>

  {/* create-libra2-dapp */}

  <TabItem label="Create Libra Dapp">
    <Steps>
      1. Instalar create-libra2-dapp

         Ejecuta el comando de abajo para instalar una dApp desde una plantilla en segundos:

         <Tabs>
           { /* npx */ }

           <TabItem label="npx">
             ```shellscript filename="Terminal"
             npx create-libra2-dapp@latest
             ```
           </TabItem>

           { /* pnpx */ }

           <TabItem label="pnpx">
             ```shellscript filename="Terminal"
             pnpx create-libra2-dapp@latest
             ```
           </TabItem>
         </Tabs>

      2. Seguir las indicaciones

         Sigue las indicaciones de la CLI para seleccionar un nombre, [plantilla](/build/create-libra2-dapp#templates), y red para tu nueva dApp.

         ![cad](~/images/cad-video.gif)

      3. ¡Comienza a construir y personalizar tu nueva dApp!

         Navega a tu nuevo proyecto y ábrelo en tu IDE favorito para continuar construyendo!

         Sigue el archivo `README.md` generado para los siguientes pasos.

      4. Continuar leyendo
