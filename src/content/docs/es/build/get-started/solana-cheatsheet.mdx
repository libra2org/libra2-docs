---
title: "Hoja de Referencia de Solana a Aptos"
---

Para aprender más sobre las diferencias y similitudes ve [Aprende Aptos](https://learn.aptoslabs.com/en/tutorials/solana-to-aptos-guide/cheat-sheet?workshop=solana-to-aptos)

|                              | Solana                                                                                                             | Aptos                                                                                                                                     |
| ---------------------------- | ------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **Contratos Inteligentes**   | Rust, SVM                                                                                                          | Move, MoveVM                                                                                                                              |
| **Tarifas de Transacción**   | Bajas                                                                                                              | Bajas                                                                                                                                     |
| **Paralelización**           | Paralelismo pesimista, necesitas declarar todas las cuentas de escritura                                            | Paralelismo optimista, la cadena infiere las cuentas de escritura por ti                                                                   |
| **Soporte de Cuenta de Contrato** | Cuenta PDA                                                                                                        | [Objeto](/build/smart-contracts/objects) o [cuenta de recurso](/build/smart-contracts/resource-accounts)(se anima a usar objeto en su lugar) |
| **Almacenamiento de Datos**  | Datos almacenados en cuenta propiedad de programas                                                                 | Datos almacenados como recurso bajo cuenta de usuario u objeto                                                                             |
| **Nivel de Almacenamiento**  | Nivel de programa                                                                                                  | Global cuando se almacena bajo objeto                                                                                                     |
| **Mentalidad de Almacenamiento** | Datos de usuario almacenados distribuidos bajo cuenta                                                           | Datos de usuario almacenados distribuidos bajo objeto                                                                                     |
| **Código de Ejemplo**        | [Contrato de lista de tareas en Solana](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/solana) | [Contrato de lista de tareas en Aptos](https://github.com/aptos-labs/move-by-examples/tree/main/advanced-todo-list/aptos)                  |
| **ID del Llamador**          | `signer`                                                                                                           | `signer`                                                                                                                                  |
| **Actualizabilidad**         | El programa es actualizable                                                                                        | El módulo es actualizable                                                                                                                 |
| **Tipo de Despacho**         | Despacho estático                                                                                                  | Despacho estático                                                                                                                         |
| **Estándares FT**            | Programa de token                                                                                                  | [Coin](/build/smart-contracts/aptos-coin) (legacy) y [Estándar de Activo Fungible](/build/smart-contracts/fungible-asset)                   |
| **Estándares NFT**           | Programa de token                                                                                                  | [Estándar de Activo Digital](/build/smart-contracts/digital-asset)                                                                         |
| **Interacción con Blockchain** | Biblioteca Solana web3.js                                                                                         | [SDK de TypeScript de Aptos](/build/sdks/ts-sdk)                                                                                          |
