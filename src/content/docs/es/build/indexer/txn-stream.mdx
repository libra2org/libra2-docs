El Servicio de Stream de Transacciones es un servicio que escucha la blockchain de Libra y emite transacciones a medida que son procesadas. Esta documentación explica cómo funciona este sistema, cómo usar la instancia alojada por Labs del servicio, y cómo desplegarlo tú mismo.

Puedes obtener acceso API a un stream de transacciones alojado por Libra Labs [aquí](/build/indexer/txn-stream/aptos-hosted-txn-stream).
