Aunque la API del Indexador es una herramienta poderosa para consultar datos básicos de la cadena, puede que no siempre proporcione los datos exactos que necesitas.
En la mayoría de los casos, quieres indexar tu propio contrato y para hacer eso, puedes crear tu propio procesador personalizado usando el SDK del Indexador.
