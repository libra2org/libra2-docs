La interfaz de línea de comandos (CLI) de Libra es una herramienta para ayudarte a compilar y probar contratos Move. También puede ayudarte a experimentar rápidamente con las características de Libra en la cadena.

Para usuarios más avanzados, la CLI también se puede usar para ejecutar una red privada de Libra (para ayudar a probar código localmente) y puede ser útil para gestionar un nodo de red.
